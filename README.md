# Moments Gallery

A simple social media clone with a React frontend and a Node.js/Express backend.

## Project Structure
- **/Backend**: Node.js server using Express.
- **/Frontend**: React application built with Vite.

## How to Run

### Backend
1. Navigate to the backend folder:
   ```bash
   cd Backend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the server:
   ```bash
   npm run dev
   ```

### Frontend
1. Navigate to the frontend folder:
   ```bash
   cd Frontend/frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
